# RosettaPad

> [!CAUTION]
> The Pico port is very early in development, nothing is promised, nothing is guaranteed, use at your own risk. 

--- 

## Planned Features

### Basic Features: 
- [ ] Basic Network Access Point/Web Panel (for basic configuration and debug)
- [ ] BT pairing (Pico ↔ DS5) 
- [ ] All buttons 
- [ ] Analog Sticks
- [ ] Analog Triggers
- [ ] USB connection (Pico ↔ PS3) 
- [ ] Battery display
- [ ] BT pairing (Pico ↔ PS3) 
- [ ] Motion controls
- [ ] PS3 Standby Wake

### Planned: 
- [ ] Profiles
- [ ] Button Remapping
- [ ] Macro's
- [ ] Lightbar color & animation customization
- [ ] Hotkeys (for macro/remapping toggles & profile switching)
- [ ] Additional controller support
- [ ] PS4/PS5 support
